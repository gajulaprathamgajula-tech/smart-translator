# 🌐 Smart Language Translator

A web-based language translation application that allows users to input text and view translated output in real time. The project focuses on building a simple and interactive user interface with basic text processing functionality.

---

## 🚀 Key Features
- User-friendly interface for text input  
- Instant display of translated output  
- Lightweight and responsive design  
- Easy to use and understand  

---

## 🛠️ Tech Stack
- HTML (Structure)
- CSS (Styling)
- JavaScript (Logic & Functionality)

---

## ⚙️ Working
The application takes user input through a text field.  
On clicking the translate button, a JavaScript function processes the input and displays the output dynamically on the screen.

---

## 📂 Project Structure
- `index.html` → Frontend structure  
- `style.css` → Styling and layout  
- `app.js` → Core functionality
